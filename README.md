# 🚢 Titanic Survival Analysis

## 📌 Project Overview
Exploratory Data Analysis (EDA) on the Titanic dataset to uncover 
patterns in passenger survival using Python and data visualization.

## 🛠️ Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## 📊 Key Findings
- Overall survival rate was ~38%
- Female passengers had a ~74% survival rate vs ~19% for males
- 1st class passengers survived at nearly 3x the rate of 3rd class
- Children under 10 had higher survival rates than adults
